# To-do List com Vue.js 3

Este projeto é uma aplicação simples de lista de tarefas desenvolvida com Vue.js, seguindo o "Guide Start Vue3". 

## Funcionalidades:
Adicionar tarefas, marcar tarefas como concluídas, remover tarefas.
HTML, CSS e Vue.js 3
